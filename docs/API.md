# Gateway API â€” Summary

Base URL: `http://localhost:8080`

## Auth
- `POST /auth/register` â€” `{ email, password }`
- `POST /auth/login` â€” sets HttpOnly cookie if success
- `POST /auth/logout` â€” clears session cookie
- `GET /auth/me` â€” returns `{ id, email, is_verified }`

## Me / Patient
- `GET /me/patient` â€” profile snapshot or `null` if not linked
- `PUT /me/patient` â€” updates demographics (whitelisted fields)

## Chat
- `POST /chat/send` â€” `{ message?, args? }` â†’ forwards to MCP tool
- `GET /chat/events` â€” SSE stream proxied from MCP

## Error Model
- Errors return JSON `{ detail: string }` and appropriate HTTP status.

## Auth Model
- Cookie name: `sid` (configurable)
- Cookies are HttpOnly; frontend must send `credentials: 'include'`.
